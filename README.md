# Scott's Food List 🍽️

Restaurant training application for Scott's Richmond.

## Features

- 🎓 **Food Trainer** - Learn dishes, wines, and service requirements
- 📱 **Mobile Catalog** - Browse all dishes with search and filtering  
- ✏️ **Food Builder** - Create and edit dishes

## Quick Start

```bash
python -m http.server 8080
```

Open http://localhost:8080

## Data

71 dishes from Scott's Food Bible including:
- Oysters & Caviar
- Starters & Mains
- Sides & Desserts

Each dish contains:
- Description
- Wine Pairing
- Glossary
- Allergens
- Kitchen MEP
- Service MEP
